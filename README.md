# Corcel

Corcel is a personal literary magazine project that I started as a class assignment, but that I hope to keep growing over time. The idea came from a simple wish: to create a digital space where poetry and literature can live freely, accessibly, and with care.

This is a first version. It is rough around the edges, and that is intentional.

---

## What is Corcel?

Corcel is a web catalogue of poetry. For now, it features ten poems by Spanish poets from the 20th century: Miguel Hernández, Gloria Fuertes, Blas de Otero, Gabriel Celaya, and Ángela Figuera Aymerich. The design is inspired by the aesthetic of personal notebooks, analogue photography, and the early internet.

The project is alive. I would like to keep adding poems, authors, and features as I learn more.

## What works and what doesn't (yet)

The decorative elements on the page (the folder, the headphones, the bunnies) are not interactive yet. The original idea was for the folder to open and reveal cards with information about the team behind Corcel. This is the next thing I would like to implement, along with smoother animations and a filter by author.

---

## How to run the project locally

1. Clone the repository
2. Install JSON Server if you don't have it:
```bash
   npm install -g json-server
```
3. Start JSON Server from the project folder:
```bash
   json-server --watch db.json
```
4. Open `index.html` with Live Server in VS Code
